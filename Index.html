The World Wide Web (WWW) is a global system of interconnected hypertext documents accessed via the internet. It allows users to navigate between web pages by clicking on hyperlinks

HTML (Hypertext Markup Language) is the standard markup language used to create and structure web pages. It provides a set of elements or tags that define the structure and content of a webpage, such as headings, paragraphs, images, links, and more.

CSS (Cascading Style Sheets) is a style sheet language used to describe the presentation of a document written in HTML. CSS controls the layout, formatting, and appearance of web pages, allowing developers to define styles for fonts, colors, spacing, and other visual aspects of the content.
